# Job_Salary_Dashboard
Built an interactive Excel dashboard to analyze salary trends by job title, country, and work type using dropdown filters and summary reporting tables

## Overview
This project is an Excel-based salary calculator and dashboard built to explore job salary trends by role, country, work schedule/type, and platform. It allows users to filter job market data and quickly view median/average salary insights and reporting visuals.

## Key Features
- Interactive salary calculator using dropdown filters (Job Title, Country, Type)
- Dashboard view with summary KPIs and charts
- Cleaned summary tables powering visuals (portfolio-ready layout)

## Tools Used
- Microsoft Excel (Tables, Data Validation dropdowns, Pivot-style summaries, charts)

## Dashboard Metrics (examples)
- Salary distributions by job title
- Salary comparison by country
- Posting volume by platform/type (if included)
